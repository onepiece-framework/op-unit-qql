QQL is Quick Query Language
===

# Supported

 * Open
 * Insert
 * Update
 * Select
 * Where
 * Join
 * Limit
 * Offset
 * Pager

# Not Supported

 * Delete
 * Where
	 * Or
 * Join
	 * Right
	 * Inner
	 * Outer

# CI

1. Copy `QQL.sqlite3` to `asset:/db/ci/` directory.

# Sandbox

1. Copy `QQL.sqlite3` to `asset:/db/sandbox/` directory.
2. Change to owner id `asset:/db/sandbox/` directory and `QQL.sqlite3`.
